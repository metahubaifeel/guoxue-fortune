# 小师傅来一卦

一个结合传统小六壬算法和现代AI智能的国学运势应用。

## 🎯 项目特色

### 传统与现代的完美结合
- **小六壬算法**: 传承千年的国学占卜智慧
- **AI智能解读**: 火山引擎豆包大模型，个性化运势分析
- **多维度结合**: 扑克牌+数字+天气+心情，全方位运势推演

### 用户体验优化
- 🎨 **精美界面**: 国风设计，优雅大气
- ✨ **流畅动画**: 水晶球加载效果，掐指一算文案
- 🎯 **智能高亮**: 重点建议内容突出显示
- 📱 **响应式设计**: 完美适配移动端

## 🚀 快速开始

### 本地运行
```bash
# 安装依赖
npm install

# 启动AI代理服务器
node ai-proxy-server.js

# 启动前端服务
npm run serve
```

访问: `http://localhost:8080`

## ✨ 核心功能

### 1. 智能起卦
- 扑克牌抽取（A-K对应1-13）
- 用户数字输入（1-99）
- 小六壬六卦推演（大安、留连、速喜、赤口、小吉、空亡）

### 2. AI运势解读
- 结合卦象、天气、心情
- 个性化建议生成
- 每次结果都不同（温度1.2）

### 3. 视觉优化
- 重要内容智能高亮
- 国风配色方案
- 响应式布局设计

## 🎮 使用流程

1. **洗牌抽牌**: 点击洗牌，选择一张扑克牌
2. **输入数字**: 输入两个幸运数字（1-99）
3. **选择天气**: 今日天气状况
4. **选择心情**: 当前心情状态
5. **获取运势**: AI大师为您解读今日运势
